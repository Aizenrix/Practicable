# RISK_REGISTER.md — Calipso.Coffee (этап 5)

Минимум 6 рисков: 3 по безопасности, 3 по эксплуатации.

## Безопасность

| ID | Риск | Причина | Вероятность | Влияние | Мера снижения |
|----|------|---------|-------------|---------|---------------|
| S-01 | Утечка `.env` | Файл случайно попал в Git | Средняя | Высокое | `.gitignore`, `security-check`, только `.env.example` |
| S-02 | Доступ к чужим данным | Нет проверки владельца заказа | Средняя | Высокое | `canAccessOrder()` в orders.routes, тесты security |
| S-03 | Уязвимая зависимость | npm audit: qs moderate | Средняя | Среднее | `deps-check`, `npm audit fix`, регулярное обновление |
| S-04 | Слабый JWT_SECRET | Демо-значение `change_me` | Высокая | Высокое | Уникальный секрет в production, не коммитить |

## Эксплуатация

| ID | Риск | Причина | Вероятность | Влияние | Мера снижения |
|----|------|---------|-------------|---------|---------------|
