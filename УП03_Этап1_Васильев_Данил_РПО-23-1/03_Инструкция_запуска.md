## Требования

| Компонент | Минимальная версия | Проверка |
|-----------|-------------------|----------|
| Node.js   | 18.x (рекомендуется 22.x) | `node --version` |
| npm       | 9.x+              | `npm --version` |
| Git       | любая актуальная  | `git --version` |

## Шаг 1. Клонирование репозитория

```bash
git clone https://github.com/Aizenrix/Practicable.git
cd Practicable
```

**Ожидаемый результат:** папка проекта с файлами `package.json`, `src/`, `public/`, `prisma/`.

## Шаг 2. Установка зависимостей

```bash
npm install
```

**Ожидаемый результат:** папка `node_modules/` создана, ошибок `ERR!` нет.

## Шаг 3. Настройка окружения

```bash
cp .env.example .env
```

Содержимое `.env.example`:

```
PORT=4000
JWT_SECRET=change_me
DATABASE_URL="file:./dev.db"
```

| Переменная | Назначение |
|------------|------------|
| `PORT` | Порт сервера (по умолчанию 4000) |
| `JWT_SECRET` | Секрет для JWT (в продакшене заменить!) |
| `DATABASE_URL` | Путь к SQLite-файлу |

## Шаг 4. Подготовка базы данных

```bash
npm run prisma:generate
npm run prisma:push
npm run prisma:seed
```

**Ожидаемый результат:** файл `prisma/dev.db` создан; в БД есть роли, столы, admin-пользователь.

## Шаг 5. Запуск тестов (опционально)

```bash
npm test
```

**Ожидаемый результат:** 8 тестов, 0 ошибок.

## Шаг 6. Запуск сервера

```bash
npm run dev
```

**Ожидаемый результат в консоли:**

```
Calipso.Coffee API запущен на порту 4000
```

## Шаг 7. Проверка работоспособности

1. Откройте в браузере: **http://localhost:4000**
2. Проверьте API: **http://localhost:4000/health** → `{"status":"ok","service":"calipso-coffee-api"}`
3. Войдите в систему:
   - Email: `admin@calipso.coffee`
   - Пароль: `admin123`

## Возможные ошибки

| Ошибка | Решение |
|--------|---------|
| `Cannot find module '@prisma/client'` | Выполните `npm run prisma:generate` |
| `Environment variable not found: DATABASE_URL` | Создайте `.env` из `.env.example` |
| `EADDRINUSE :::4000` | Порт занят — смените `PORT` в `.env` |
| `Error: P1001` (Prisma) | Выполните `npm run prisma:push` |

## Остановка сервера

Нажмите `Ctrl+C` в терминале, где запущен `npm run dev`.
